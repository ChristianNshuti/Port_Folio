# My React App

This is a React application built with Tailwind CSS and Framer Motion. It features a Navbar, Hero section, About section, Projects, and Skills.

## Features

- Responsive full-width layout
- Smooth scrolling to sections
- Animated components using Framer Motion
- Tailwind CSS for styling
- Interactive projects and skills showcase

## Getting Started

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/ChristianNshuti/Port_Folio
cd Port_Folio
